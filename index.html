IA Generated applications

Generation process

Each application is generated from a markdown file which content is generated by ChatGPT 4.

The generation uses Smol Developer an AI Junior developer that allows you to generate a simple application from a markdown file and processing using Modal a cloud computing platform.

There is no human intervention in the generation process and result, the only human intervention comes from the prompt request made to ChatGPT.
